## Overview

The **Sales & Profit Dashboard** was created in Excel to provide a clear and interactive view of business performance. It combines summary KPIs and visual analysis to help users understand sales patterns, profit performance, customer activity, and geographic trends in one place.

The dashboard is designed to be simple, clean, and decision-friendly, making it useful for reporting, portfolio projects, and business analysis practice.

## Key Features

- **Total Sales KPI** for overall revenue tracking
- **Total Profit KPI** for quick profitability insights
- **Profit by Year** chart to compare yearly profit performance
- **Sales by Category** visualization for product/category comparison
- **Customer Count by Year** donut chart for customer trend analysis
- **Sales by State** map chart to identify regional performance
- **Monthly Sales Trend** chart to analyze seasonality and monthly movement
- **Top 5 Customers by Profit** chart to identify the most profitable customers
- **Interactive slicers** for filtering dashboard data dynamically
- **Refresh option** to update pivot-based visuals when data changes
